A visual demonstration of how Project Mulla works

Project Mulla is a MPESA API RESTful mediator. It lets you make familiar HTTP REST requests, transforming your requests to the fiddling dreaded SOAP/XML requests. It then communicates with the Safaricom MPESA G2 API gateway, transforming all SOAP responses from the SAG to RESTful JSON responses that you then consume effortlessly.

The Middleman

After installing and starting Project Mulla, it will route every incoming request from your server/client with the specified public DNS to the Safaricom SAG. And every response, after processing the the SOAP body, to your server/client.
