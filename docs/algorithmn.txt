Remove the XML header tag => /\<\?[\w\s\=\.\-\'\"]+\?\>/gmi

Extract 1st line using this => /(\<([\w\-]+\:[\w\-]+\s)([\w\=\-\:\"\'\\\/\.]+\s?)+?\>)/gi

Get the PREFIXES using this => /([\w\-]+\:[\w\-]+)+/gi
They start with "xmlns:bla-bla"

Modified, much specific => /((xmlns):[\w\-]+)+/gi or /(xmlns\:)([\w\-]+)/gi

Get the "bla-bla:" part and replace it with "" (nothing);

Now we have a clean HTML-like tree. The parse5/minidom library can take over from here
